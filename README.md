# TurnIQ Prototype

This is a simple Next.js version of the TurnIQ prototype.

## Run locally

1. Install dependencies

```bash
npm install
```

2. Start the app

```bash
npm run dev
```

3. Open

```bash
http://localhost:3000
```

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to Vercel
3. Create a new project from the repo
4. Click Deploy
